# FACE-RECOGNITION-ATTENDANCE-SYSTEM
Face Recognition Attendance System is a Python-based computer vision application that automates attendance tracking using real-time facial recognition. Built with OpenCV and facial encoding techniques, the system accurately identifies registered users and records attendance with timestamps, eliminating manual attendance processes.

Overview

The Face Recognition Attendance System is an automated attendance management application that uses computer vision and facial recognition technology to identify individuals and record their attendance in real time. The system captures facial features through a webcam, compares them with stored facial data, and marks attendance automatically, reducing manual effort and improving accuracy.

Features

Real-time face detection and recognition
,Automated attendance marking
,Secure and accurate identification
,Attendance records stored in CSV or database format
,Easy registration of new users
,Reduces proxy attendance and human errors

Technologies Used

Python
,OpenCV
,face_recognition Library
,NumPy
,Pandas
,CSV / SQLite Database

Working Principle

Capture images of registered users.
Extract and encode facial features.
Store face encodings in the system database.
Access the webcam for real-time video input.
Detect and recognize faces from the live video stream.
Match detected faces with stored encodings.
Automatically record attendance with date and timestamp.
Save attendance records for future reference.
